# Calculadora Sygna - Planes y Funcionalidades 2025

Calculadora modular interactiva para calcular el coste de los planes y módulos de Sygna.

## Características

- ✅ Selección de base (Micro, Pyme, Empresa)
- ✅ Selección de módulos funcionales con validación de dependencias
- ✅ Cálculo en tiempo real del coste total
- ✅ Gestión de extras (usuarios adicionales, facturas adicionales, etc.)
- ✅ Generación de presupuesto en formato HTML imprimible
- ✅ Diseño responsive y moderno

## Uso

1. Abre `index.html` en tu navegador
2. Selecciona una base (Micro, Pyme o Empresa)
3. Activa los módulos que necesites
4. Configura los extras si es necesario
5. El coste total se calcula automáticamente
6. Haz clic en "Generar Presupuesto PDF" para crear un presupuesto imprimible

## Módulos Disponibles

- **Facturación Digital**: Con diferentes paquetes (S, M, L, XL, Enterprise)
- **RRHH**: Gestión de empleados y control horario
- **Nóminas**: Cálculo de nóminas (requiere RRHH)
- **Tesorería**: Gestión de flujo de caja
- **Contabilidad**: Plan contable y balances (requiere Facturación)
- **Documentos y Firma Digital**: Repositorio y plantillas
- **Dashboards e Integraciones**: Paneles y KPIs
- **Tickets**: Sistema de incidencias

## Dependencias

- La Contabilidad requiere Facturación activa
- Las Nóminas requieren RRHH activo

## Tecnologías

- HTML5
- CSS3 (con diseño responsive)
- JavaScript (Vanilla)
